# GeminiLive AI Assistant

## Overview

GeminiLive AI Assistant is an AI-powered chatbot built using Python, Flask, and Google Gemini API.

## Features

* Google Gemini Integration
* Conversation Memory
* Real-Time Weather Information
* Bitcoin Price Lookup
* Google Search Integration
* Chat Export Functionality
* Modern Chat Interface

## Tech Stack

* Python
* Flask
* Google Gemini API
* OpenWeather API
* CoinGecko API
* HTML
* CSS
* JavaScript

## Project Structure

* Frontend: HTML, CSS, JavaScript
* Backend: Flask
* AI Model: Google Gemini
* Memory Storage: JSON

## How to Run

1. Clone repository
2. Create virtual environment
3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Add API keys in .env file
5. Run

```bash
python app.py
```
